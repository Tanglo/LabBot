Detailed Description

A class for displaying data as a moving bar indicator.

This class draws a bar, and optionally a target line, that are specified by class properties.

Method Documentation

- (void) setCalibratedTarget: (CGFloat)  newTarget

This methods sets the target property using the formula:

target = gain*newTarget +offset

where target, gain, and offset are properties of the receiver.

Parameters
newTargetA CGFloat that will be calibrated and set as the new target of the receiver
- (void) setCalibratedValue: (CGFloat)  newValue

This methods sets the value property using the formula:

value = gain*newValue +offset

where value, gain, and offset are properties of the receiver and have been previoosuly set.

- (void) twoPointLinearcalibrationUsing: (NSPoint)  firstPoint
And: (NSPoint)  secondPoint 

Sets the gain and offset properties of the receiver based on a two-point linear calibration between two NSPoint values.

Parameters
firstPointThe x-value of this point is the raw uncalibrated value, the y-value is the calibrated output between 0.0 and 1.0. The y-value of this point should be less than the y-value of secondPoint.
secondPointThe x-value of this point is the raw uncalibrated value, the y-value is the calibrated output between 0.0 and 1.0. The y-value of this point should be more than the y-value of firstPoint.
