Detailed Description

A top level class for interacting with data aqusition hardware.

This is a top level class that implements properties and methods common to using data aquisition devices. Generally this class will be subclassed to add properties and methods spepcific to you device or class of device. For example, DRHLabJack is a subclass of DRHDaqDevice and adds properties and methods specific to LabJack devices. DRHLabJackU6 is a subclass of DRHLabJack and adds the properties and methods specific to the LabJack U6 device.

- (NSArray *) readStreamData

It is compulsory for subclasses to override this method so that all DRHDaqDevice classes are compatible with DRHSamplingOperation.

Reads the buffered data from the receivers device and returns it stored in an array.

Returns
An array of the data collected from the device buffer.

Reimplemented in DRHLabjackU6, and DRHLabJack.
