Detailed Description

A class for controlling the main window of an experiment.

A DRHExperimenterWindowController is intended to be used to control the main window of an experiment. The class is intended to the viewController class in an MVC setup that uses DRHExperimentData and so it includes IBOutlet instances that match the instance variables of the DRHExperimentData class. It is compulsory to use this class if you use the DRHExperimentDocument class.

If subclassing DRHExperimentWindowController in Swift it can be easier to override windowNibName instead of overriding initailisers. For example:

override var windowNibName : String! {
return "MyWindowControllerNib"
}
