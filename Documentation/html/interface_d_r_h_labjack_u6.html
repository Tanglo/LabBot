Detailed Description

A class for controlling a LabJack U6 or U6-Pro data aquisition device.

This class provides properties and methods to facilitate interacting with a LabJack U6 or U6-Pro device. This class is specific to the U6 and U6-Pro. If you have a different LabJack device you should be using a different subclass of DRHLabJack. If your data aqusition hardware is not a LabJack device you should be using a different subclass of DRHDaqDevice.
