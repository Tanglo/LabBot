Detailed Description

An root level object to store basic experimental data and tables of experimental variables.

A DRHExperimentalData object is intended to be the top level model object in an experimental applcaition. It will contain all experimental data. A DRHExperimentalData object contains instance variables to store basic experiment details such name, data, session, etc and also contains one instance of DRHDataMatrix. It is expected that most LabBot experiments will use a sub-class of DRHExperimentalData because each experiment is unique and will have it's own requirements. This object is here to provde a base template to facilitate defining a LabBot experiment's root data object.
