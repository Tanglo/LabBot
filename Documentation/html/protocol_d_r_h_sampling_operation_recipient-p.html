Detailed Description

A protocol that defines which methods a class must implement to receive data from a DRHSamplingOperation.

Classes that will act as the dataRecipient of a DRHSamplingOperation must implement this protocol.

Method Documentation

- (void) receiveData: (NSArray *)  newData

This method will be called by an instance of a DRHSamplingOperation everytime data is streamed from a DRHDaqDevice. This method receives an array containing the most recently streamed data.

Parameters
newDataThis array contains the streamed data. The format of the data within this array is determined by the readStreamData method of the DRHDaqDevice subclass that read the data from the hardware.
